## Description

Junt is a Chrome extension that helps you to track your job applications. It's totally free and open source.

## Development

```bash
pnpm build:dev
pnpm serve
```

* Open `chrome://extensions/`
* Turn `Developer mode` on
* `Load unpacked`
* Select `dist` dir in the project root
* Pin `Junt`
